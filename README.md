# MightyLauncher
The Mighty Doom server files with a simple launcher, Mighty Doom shut down on August 7th 2024, this server is supposed to be a self hosted, with a patched APK, you can get the game in a playable state again. By the end of the setup, you will have a Mighty Doom server hosted on your computer, with the option to have the server run on boot/login. Credit to ShackleBolt13 for the original server, follow their guide to set up the server. https://github.com/Shacklebolt13/mightydoom-gameserver
